AI Prompt Guide

Use Claude or ChatGPT to build a custom CSV
Got a complex labelling request that doesn't fit nicely into the manual or bulk paste boxes? Paste the prompt below into Claude or ChatGPT, fill in your locations, and it will spit out a CSV you can save and drop into the Import CSV tab.

The CSV Format

SmartLabel reads CSVs with two columns. One row per printed label.

Location,Arrow
  • Location — e.g. K2-02-018-1 (prefix-aisle-bay-level) or K2-EW-001 (3-part).
  • Arrow — UP, DOWN, or NONE. Set explicitly per row — never inferred from the location code.
  • First row is the header literal: Location,Arrow
  • One label per row. No blank lines.

Copy this prompt

I need a CSV for SmartLabel, a warehouse label generator. The CSV must have: - Header row: Location,Arrow - One label per row - Location format: PREFIX-AISLE-BAY-LEVEL (e.g. K2-02-018-1) where AISLE is 2 digits, BAY is 3 digits, LEVEL is 1 digit - Some prefixes use 3 parts: PREFIX-ZONE-NUMBER (e.g. K2-EW-001) for end-of-row / ceiling labels - Arrow values: UP, DOWN, or NONE Generate a CSV for the following warehouse layout: <-- DESCRIBE YOUR LOCATIONS HERE. Examples: - All bays in aisle K2-02 (bays 1-100), levels 1-3 - Even-numbered bays only in K2-03, bays 18-60, level 1 only - Single locations: K2-04-001-1, K2-04-005-1 - End-of-row K2-EW-001 through K2-EW-050 with no arrows --> Arrow rules: - Level 1: DOWN - Level 2: UP - Level 3: UP - End-of-row (3-part codes): NONE Output ONLY the raw CSV content with no markdown, no commentary, no code fences. Start the response with the literal "Location,Arrow" header line.
How to use: click Copy, paste into Claude or ChatGPT, edit the <-- DESCRIBE YOUR LOCATIONS HERE --> block with your actual layout, send. Save the response as my_labels.csv and drag it into the Import CSV tab.

Tips for better output

  • Be specific about ranges. "Bays 1 through 50" beats "first half".
  • Spell out filters. "Even bays only" or "every other bay starting at 18".
  • List levels explicitly. "Levels 1, 2, and 3 for each bay" vs "level 2 only".
  • Mention exceptions. "Skip bay 25" or "bay 50 should have NONE arrow".
  • Verify the count. After generating, ask the AI: "How many rows did you produce?" and sanity-check it.

Quick example

Replace the description block with something like:

All bays in aisle K2-02 from 001 to 100, levels 1-3. Then bay K2-04-072-2 only. Then K2-EW-001 through K2-EW-050 (no arrows).

The AI will produce a CSV with ~352 rows ready to import.
